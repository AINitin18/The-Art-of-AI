## Welcome

Welcome to the official GitHub repository for **The Art of AI**.

This repository contains the **code examples, tiny datasets, mini-projects, exercises, and hands-on implementations** that support the book. The goal of this repository is to help learners move from reading AI concepts to actually running, changing, and understanding working AI code.

AI is often taught as a collection of complex algorithms, formulas, and tools. **The Art of AI** takes a different path. It explains AI as an art of thinking with data, reasoning with uncertainty, learning from patterns, and building intelligent systems responsibly.

---

## What This Book Helps You Learn

The book is designed to help you build a strong practical foundation across four important areas:

### Statistics & Probability
Understand data, uncertainty, distributions, probability, and the basic thinking required before building AI models.

### Machine Learning
Learn how machines identify patterns, make predictions, classify data, and improve from examples.

### Deep Learning
Understand neural networks, hidden layers, activation functions, training, validation, and prediction using simple examples.

### Generative AI
Explore modern AI systems that can generate text, assist users, answer questions, summarize content, and support real-world applications.

## How to Use This Repository with the Book

Use this simple method for every chapter:

1. Read the chapter from the book.
2. Understand the **Why → What → How** explanation.
3. Open the related notebook or script from this repository.
4. Run the code as-is first.
5. Change one input value, one dataset row, or one parameter.
6. Run the code again and observe the result.
7. Repeat until the concept becomes clear.

This is one of the fastest ways to build real AI understanding.

## Getting Started

### 1. Clone this repository

```bash
git clone https://github.com/<your-github-username>/the-art-of-ai.git
cd the-art-of-ai
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

### 3. Activate the virtual environment

For Windows:

```bash
venv\Scripts\activate
```

For macOS/Linux:

```bash
source venv/bin/activate
```

### 4. Install required libraries

```bash
pip install -r requirements.txt
```

If `requirements.txt` is not available yet, you can start with:

```bash
pip install numpy pandas matplotlib scikit-learn tensorflow torch notebook
```

### 5. Start Jupyter Notebook

```bash
jupyter notebook
```
